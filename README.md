# 線上會議報名表

## 如何啟動

```bash
npm install
npm run dev
```

若要產出生產版本：

```bash
npm run build
```

檢視生產版本：

```bash
npm run preview
```

## 使用技術

- Vite
- React 19
- TypeScript
- Tailwind CSS
- React Hook Form
- Zod
- @hookform/resolvers/zod

## 技術選擇原因

- TypeScript：提供型別，減少執行時錯誤，改善代碼可維護性。
- Tailwind CSS：快速完成一致、可維護的 UI。
- React Hook Form + Zod：集中處理表單狀態、驗證與條件欄位邏輯。
